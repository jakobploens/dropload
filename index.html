Dropload

A bunch of tiny (3 kb minified) components for jQuery to handle drag and drop file uploads, focused on progessive enhancement and minimalism.

Download | Github Repo

Honest readme: it's only tested in latest Safari, Chrome and Firefox so far. It's rough. Not very well documented. Basically this page is all there is. I'm not a JS guru/ninja, so this does not include the latest frontend bells and whistles. Feel free to play with it! Please don't expect support. Open source is hard! Read more about the license.

$.filereader

A helper to simplify the FileReader Api

$.upload

A helper to simplify XHR2 file uploads

Built-in support flags: $.support.xhr2, $.support.formData, $.support.upload

$.fn.drop

Prepares an element to receive drops (Demo)

$.fn.filedrop

Combines $.fn.drop and $.filereader (Demo)

$.fn.dropload

Combines 'em all to provide a nice and clean drag and drop upload area. (Demo)

Built-in support flags: $.support.dropload
